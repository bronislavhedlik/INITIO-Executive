# Open Decisions

Executive Reality Review neautorizuje novou produktovou prioritu před dokončením Product Reality Synchronization.

| Třída | Otevřená otázka | Autorita / potřebná evidence |
| --- | --- | --- |
| `Unknown` | Kdo je Owner, Reviewer, Approver a Publisher firemních a brandových oblastí? | Výslovné rozhodnutí vlastníka + autoritativní registry. |
| `Unknown` | Jaká je schválená mission, vision, portfolio a firemní priorita po Foundation? | Approved artefakty v INITIO. |
| `Unknown` | Která jediná produktová priorita skutečně následuje? | Synchronizovaný aktuální stav `BH_OS`, `bh-os-clean`, runtime evidence a platná LOCK rozhodnutí; historické Product Cases nestačí. |
| `Unknown` | Je GA4 Connection Layer dokončený, nebo pouze implementovaný a testovaný? | Aktuální completion decision + runtime Reality Review + případný autoritativní LOCK. |
| `Unknown` | Jaký je aktuální finální stav Google Ads Connection Layer? | Sjednotit starší `Not Production Ready`, novější dílčí PASS a aktuální `PARTIAL` acceptance evidence. |
| `Unknown` | Jaký je aktuální finální stav Google Merchant Center? | Uzavřený provider Reality Review, isolation checks a odstranění/akceptace zbývajících findings. |
| `Unknown` | Je Production Readiness dokončena po stavu `READY FOR CTO CERTIFICATION`? | Finální CTO/release rozhodnutí a aktuální runtime validation/smoke evidence. |
| `Unknown` | Jaký je finální scope BH OS V1, RBAC, retention a případný další connector order? | Product Owner decisions + Product Bible/Product Brain po Reality Synchronization. |
| `Unknown` | Je Operating Mind V1 production base nebo in development? | Jednotná aktuální stavová evidence v BH_OS. |
| `Unknown` | Má Product Strategy formální LOCK? | Oficiální verze, datum, approval/lock a changelog. |
| `Unknown` | Jaké jsou reopening conditions u neúplných BH_OS LOCKů? | Doplněný autoritativní LOCK register. |
